# Recommender Systems Research

## Описание

Эта проект посвящённый сравнению различных алгоритмов рекомендательных систем.
В работе реализованы несколько подходов к построению рекомендаций и проведено их сравнение по стандартным метрикам качества.

В качестве датасета используется **MovieLens 1M**.

Цель проекта — сравнить разные классы рекомендательных алгоритмов в одинаковых условиях обучения и тестирования.

---

## Использованные алгоритмы

В проекте реализованы следующие методы:

* Popularity recommender
* Content-based filtering
* User-based collaborative filtering
* Item-based collaborative filtering
* Matrix factorization (SVD)
* Matrix factorization (NMF)

Все алгоритмы обучаются на одном train/test split и сравниваются по одинаковым метрикам.

---

## Метрики

Для оценки качества используются следующие метрики:

* Precision@K
* Recall@K
* NDCG@K
* RMSE

Релевантные объекты определяется как:

rating ≥ 4

Метрики считаются по каждому пользователю отдельно и затем усредняются.
